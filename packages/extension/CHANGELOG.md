# Changelog

## 0.3.0

- Canonicalized bridge conversation endpoints under `/conversations`
- Removed legacy conversation HTTP routes with no compatibility aliases
- Switched `antigravity-ask ask` to the headless conversation path
- Enabled real `--variant` support for both `ask` and `send`
- Updated OpenAPI, README, and skill documentation to match the new API

## 0.1.0

- First public preview release of Antigravity Ask Bridge.
- Added local HTTP and WebSocket bridge servers for Antigravity chat automation.
- Added native command and LS bridge endpoints for conversation control and artifact access.
